# Linux

## Linux Commands learnt today

1. **pwd** - present working directory.
            Prints the location of the current directory.

2. **cd directory_name** - change directory.
                           Opens the mentioned directory. Variants of this command are **cd -** (moves to the last opened directory) and **cd ..**(moves up by one level that is the immediate parent directory of the direcotry currently open)

3. **ls** - list.
           Displays the list of everything present in the directory. Variants of this command are **ls -l** (displays a list which contains details about each and every item like name, size, last time of modification, etc.), **ls-a** (displays the hidden items in the directory), **ls -h** (displays stuff in human understandable language). These variant commands can be used in combinations like **ls -lh**, **ls -al**, etc.

4. **mkdir** - make directory.
               Creates a new directory in the present directory.

5. **clear** - clears all the output shown on Terminal.

6. **touch file_name1.txt file_name2.txt** - Creates text files in the current directory.

7. **nano file_name.txt** - Nano is a text editor. Using the nano command opens the text file in the Nano text editor.

8. **cat file_name.txt** - This displays the whole text file (basically dumps the whole thing on you).

9. **less file_name.txt** - This command displays the text file page-wise ( easily maneuverable and quits on **q**).
